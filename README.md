# Nassau Candy Distributor — Shipping Route Analysis

Ngl, when I first got this project, I had no idea 
how to even approach a logistics dataset.

Spent time actually understanding the data, built 
everything step by step, and this is where I landed.

## What This Project Is

A Power BI dashboard analyzing factory-to-customer 
shipping routes for Nassau Candy Distributor — 
10,194 orders, 5 factories, 196 routes across the US.

The goal was simple: figure out which routes are 
efficient, which ones are bottlenecks, and why.

## What I Built

- 4-page interactive Power BI dashboard
- Power Query transformations (custom columns, 
  date fixes, factory mapping)
- 8 DAX measures for KPIs
- Route efficiency ranking (Top 10 / Bottom 10)
- Geographic delay analysis by state
- Synced slicers across all pages

## Key Findings (the interesting stuff)

- Same Day shipping averages 0.38 days — 
  Standard Class takes 5.32 days
- First Class has a 15% delay rate — higher than 
  Second Class (7%), which honestly surprised me
- New Mexico and Minnesota are the biggest 
  bottleneck states (16% delay rate each)
- Lot's O' Nuts handles 55% of all orders with 
  the best efficiency among high-volume factories

## Tech Used
